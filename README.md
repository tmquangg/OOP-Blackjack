# OOP Blackjack

## Features

* Hit / Stand
* Dealer AI
* Tự động tính điểm
* Thắng / Thua
* Đồ họa làm từ SDL3

## Yêu cầu

* Visual Studio
* SDL3
* SDL3_image

## Cách chạy file

1. Mở Blackjack.sln
2. Build Solution
3. Chạy

## Ko có SDL trên máy

Có 1 file thuần code riêng để chạy game blackjack trên console
